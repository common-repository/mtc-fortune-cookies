== Description ==

This plugin allows you to create and edit fortune cookies that will appear in a widget that you can place on your site.

Once installed,just click on the "Fortune Cookies" menu to add or edit the fortune cookies.

The widget will randomly pick from a single active fortune cookie each time the page is loaded.

== Installation ==

1. Upload all of the files to the /wp-content/plugins/mtc-fortune-cookies/ folder
2. Activate the plugin through the 'Plugins' menu in WordPress
3. A new menu will appear in the Admin back end.  Click on it
4. Add Fortune Cookies to the system by clicking on the "Add New" button at the top
5. Go to the Widgets screen
6. Drag the Fortune Cookies widget into one or more of your sidebars
